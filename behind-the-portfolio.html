Detrás del portafolio

Decisiones de diseño, estructura y experiencia de usuario

Proyecto personal de diseño y desarrollo frontend

Este portafolio fue diseñado como un sistema visual para presentar proyectos de forma clara, consistente y responsiva. El objetivo principal fue construir una experiencia fluida donde el contenido y la navegación fueran intuitivos.

Más que un sitio de presentación, este proyecto funciona como un ejercicio de estructura frontend y diseño de interfaz aplicado.

Paleta visual

Uso de fondo oscuro con acentos en color para generar contraste y dirigir la atención del usuario.

La combinación de colores busca equilibrio entre estética moderna y legibilidad, utilizando tonos oscuros como base y acentos para elementos interactivos.

Esto permite que los proyectos y llamados a la acción tengan mayor jerarquía visual sin saturar la interfaz.

Estructura y jerarquía

Organización basada en grid para mantener consistencia en la presentación de proyectos.

La estructura se diseñó para facilitar la exploración visual del contenido, priorizando la claridad sobre la complejidad.

Cada sección está pensada como un bloque independiente dentro de un sistema modular.

Experiencia de usuario

Navegación fija y transiciones suaves para mantener continuidad en el recorrido del usuario.

Se incorporaron microinteracciones en elementos clave como botones y tarjetas para reforzar la respuesta visual del sistema sin distraer del contenido.

El sitio combina estructura de landing page con navegación multipágina, permitiendo explorar secciones sin perder el contexto general del portafolio.

Optimización y rendimiento

Estructura ligera con recursos optimizados para mantener tiempos de carga bajos.

Se trabajó con imágenes comprimidas y formatos modernos para equilibrar calidad visual y rendimiento, priorizando la velocidad de carga en dispositivos móviles.

Conclusión

Este portafolio representa una forma de construir interfaces con intención: donde el diseño no solo organiza contenido, sino que guía la experiencia.

Más que una presentación de proyectos, es una demostración de cómo se traduce el pensamiento de diseño en estructura, claridad y comunicación visual.
